# Fraud Investigation Case Study

## Overview
This project simulates a real-world fraud investigation conducted by a financial risk analyst. The objective is to analyze transaction data to identify patterns associated with fraudulent activity and uncover potential risk indicators.

## Objectives
- Analyze transaction behavior
- Identify patterns linked to fraudulent activity
- Detect suspicious transactions
- Provide recommendations for fraud prevention

## Tools Used
- Python
- Pandas
- Matplotlib
- Seaborn
- Jupyter Notebook

## Dataset
The dataset contains anonymized credit card transactions including transaction time, amount, and labeled fraud cases.
